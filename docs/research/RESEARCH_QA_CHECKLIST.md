# SAFE Research QA Checklist

Utiliser cette checklist avant de considerer une recherche comme diffusable.

## Verification des faits

- [ ] La question de depart est precise
- [ ] Les dates de recherche sont affichees
- [ ] Les chiffres importants sont verifies une deuxieme fois
- [ ] Les noms propres sont orthographies correctement
- [ ] Les exigences legales ou reglementaires sont rattachees a des sources fiables
- [ ] Les citations ne deformant pas le sens original

## Verification des sources

- [ ] Les sources primaires ont ete privilegiees
- [ ] Les liens sont encore valides
- [ ] Les sources secondaires sont identifiees comme telles
- [ ] Aucune affirmation critique ne repose sur une seule source faible

## Verification du raisonnement

- [ ] Les inferences sont separees des faits verifies
- [ ] Les zones d'incertitude sont marquees `A_CONFIRMER`
- [ ] Les recommandations decoulent bien des faits presentes
- [ ] Les contradictions entre sources sont signalees

## Verification redactionnelle

- [ ] Le resume executif est intelligible seul
- [ ] Les sections suivent une progression logique
- [ ] Le texte est dense mais lisible
- [ ] Il n'y a pas de remplissage ou de repetition sterile

## Verification d'usage

- [ ] Le lecteur sait quoi faire apres lecture
- [ ] Les risques sont explicites
- [ ] La recherche peut servir de base a une spec, une decision ou un delivery
